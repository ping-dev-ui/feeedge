x-distribution-log.md, then logs it. All text is dash free and in the lowercase
Edie voice. Post the caption text exactly as written.

Two kinds of post:
- If a post has an "Image:" line, upload that local image file into the X composer
  (it is a branded chart, no person). With an image attached, X shows the image,
  which is intended. Post the caption text too (the feeedge.com link stays as plain
  text in the caption, that is fine).
- If a post has NO "Image:" line, just post the caption text and let the feeedge.com
  link unfurl into its card as before.

When the queue is exhausted, the task posts nothing and notifies Miguel to add more.
